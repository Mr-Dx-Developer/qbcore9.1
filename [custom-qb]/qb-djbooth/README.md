# qb-djbooth
Play music on configured coords!

## Dependencies
* [xsound](https://github.com/Xogy/xsound)
* [PolyZone](https://github.com/qbcore-framework/PolyZone)
* [qb-menu](https://github.com/qbcore-framework/qb-menu)
